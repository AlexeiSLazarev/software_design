public int findMax(int[] arr) {
    int max = arr[0]; 
    for (int i = 1; i < arr.length; i++) {
        if (arr[i] > max) {
            max = arr[i];
        }
    }
    return max;
}

Инвариант цикла:
На каждой итерации i, переменная max содержит максимальное значение среди элементов массива от 0 до i-1.

Доказательство:
Перед началом цикла max содержит элемент с индексом 0, он же и максимальный, т.к. единственный.
На каждом последующем шаге цикла, вычисляется максимальный из двух элементов - содержащимся в max и i-ом
и максимальное из двух значенияп присваивается в max. На последнем шаге происходит сравнение max с последним 
элементом массива.
Соответственно для любого шага до i-1 max содержит максимальное значение среди пройденных элементов массива.
